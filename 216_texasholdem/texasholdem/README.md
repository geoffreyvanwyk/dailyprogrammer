# Texas Holdem Poker

Deals cards to variable number of players as in Texas Holdem Poker.

## Usage

    $ python setup.py develop
    $ texasholdem

## Output

<pre>

#               TEXAS HOLDEM POKER               #
#================================================#

#==================== HANDS =====================#

Player3: 10♥ 5♦

#================== COMMUNITY ===================#

#================================================#
</pre>
